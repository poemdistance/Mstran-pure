## Features
Read the source input from terminal and auto detect the language (Chinese or English), requesting the translate result from fanyi.baidu.com and then crawling the json data from its server. Finally puts something might be useful displaying in the terminal.

## Install
    $ pip3 install -r requirements.txt  #You might need to install pip3 firstly
    $ sudo ./setup.py install           #Need to be executed under the root of project

## Usage
 After finish the installation. You can type following command to interact with terminal to get the translation result:

    $ bdtran
    > be
